📚 Hjelp - Set Sail

🚀 Hurtigstart

  1. Klikk i kartet for å sette waypoints (minst 2)
  2. Velg båt i polar-seksjonen (eller last ORC-database)
  3. Sett avgangstidspunkt og motorfart
  4. Klikk Beregn - få full analyse

🗺 Kartvalg

Klikk lagikonet øverst til høyre i kartet for å velge mellom:

  • Kartverket Sjøkart - offisielle norske sjøkart (anbefalt)
  • Kartverket Topo - topografisk kart
  • Esri Ocean - internasjonalt havkart
  • Esri Satellitt - satellittbilder
  • OpenStreetMap - åpent kart
  • CartoDB - lyst kart
  • OpenSeaMap-overlay - sjømerker (kan kombineres)

⛵ Båt og polardiagram

  • Innebygd: Dufour 40 Performance og J/109
  • ORC-database: 1900+ norske båter med ekte VPP-data
  • Last .pol-fil: Egen polardiagram i ORC-format

🎯 Motor- og seil-logikk

  • Motorfart = 0 (blank): Kun seil
  • Motorfart > 0: Motor brukes hvis seilfart < min seilfart
  • Min seilfart: Terskel for når motor starter automatisk
  • Maks segment NM: Lange etapper deles for nøyaktig analyse

💡 Jibe/Slag-tips

Grønne stiplete linjer = anbefalt jibe (med vinden)
Røde stiplete linjer = anbefalt slag (mot vinden)

Tipsene vises kun ved klart kryss (TWA < 40°) eller dyp lens (TWA > 150°) med minst 10% gevinst. Bruk checkboksen "Vis VMG-baner" for å skru av/på.

🌐 Isochroner

Tegner hvor langt du kommer i 1h, 2h, 3h osv. basert på vind og polar. Tar hensyn til land og grunne områder.

🌬 Værvindu-finder

Tester +/- 24 timer fra avgangstidspunktet for å finne raskeste avgang. Klikk en rad i resultatlisten for å velge den avgangen.

🌊 Tidevann, strøm og bølger

  • Tidevann: Fra Kartverket (sehavniva.no)
  • Strøm: Fra Open-Meteo Marine API
  • Bølger: Fargekodet etter høyde (grønn/oransje/rød)

🏔 Landunngåelse

Algoritmen bruker piksler fra Kartverkets offisielle sjøkart for å detektere:

  • 🟡 Land (gule områder)
  • 🌫 Tidevannssoner (grønne)
  • 🟦 Grunne områder (lyseblå)
  • ⚪ Trygt seil-vann (hvit)

⌨ Tastatursnarveier

  • F11: Fullskjerm
  • F12: Utviklerverktøy (for feilsøking)

💾 Lagring av ruter

Skriv navn i feltet og klikk "Lagre". Ruter lagres lokalt i nettleseren og kan lastes senere med alle innstillinger.

⚠ Viktige forbehold

  • 🚤 Ikke et erstatning for offisielle sjøkart eller GPS
  • 📊 Polardata er estimat - bruk ORC-sertifikat for nøyaktighet
  • 🌬 Vind-/strømdata kan være forsinket eller mangle
  • ⚓ Lokalkunnskap og sjømannskap er nummer én
  • 🌊 Tidevannsdata krever Kartverket-API som av og til er nede

Bruk appen som beslutningsstøtte, ikke autopilot!

📡 Datakilder

  • Sjøkart: Kartverket (kartverket.no)
  • Vind/Strøm/Bølger: Open-Meteo (open-meteo.com)
  • Tidevann: Kartverket Sehavniva
  • Polar-database: ORC (data.orc.org) og jieter/orc-data
  • Vind-visualisering: Windy (windy.com)
